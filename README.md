1- Make sure the `dist/` and `bin/` subdirectories exist on the root of the project

2. to bun the app locally : 
```bash
bun start
```

3. to package the app : 
```bash
bun run package
```

4. to make the app : 
```bash
bun run make
```

5. to publish the app to github releases: 
```bash
bun run publish
```
